# 🖼 Smart Image Studio

## Project Objective

Smart Image Studio is a desktop-based image processing application developed using Python and CustomTkinter. The project provides a simple and user-friendly interface for performing common image editing and enhancement operations. It allows users to edit images without using professional software while learning the fundamentals of digital image processing.

---

## Tools and Technologies Used

- Python 3.12
- CustomTkinter
- Tkinter
- Pillow (PIL)
- NumPy

---

## Project Modules

| Module | Description |
|--------|-------------|
| gui.py | Main graphical user interface |
| adjust.py | Brightness, contrast and sharpness adjustment |
| config.py | Application configuration and settings |
| display.py | Image display functions |
| filters.py | Image filtering operations |
| history.py | Undo and Redo functionality |
| icons.py | Icons used in the application |
| image_processing.py | Core image processing functions |
| input_output.py | Open and Save image operations |
| layout.py | GUI layout management |
| sidebar.py | Sidebar containing editing tools |
| toolbar.py | Top toolbar controls |
| utils.py | Helper and utility functions |
| zoom.py | Zoom In and Zoom Out functionality |

## How to Run the Project

### Step 1
Install Python 3.x.

### Step 2
Install the required libraries.

```bash
pip install customtkinter pillow numpy
```

### Step 3
Open the project folder in Visual Studio Code or any Python IDE.

### Step 4
Run the application.

```bash
python main.py
```
